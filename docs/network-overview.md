# StemLab Network – Overview

This document describes the network for the 2025–2026 Wiesbaden Stemlab.

## Devices
- SillyRouter – C1112 Platform – WAN edge + DHCP/NAT
- SillySwitch – WS-C3750X-48P – Core/Access switching

## VLANs
- VLAN 10 – Lab – 172.16.10.0/24
- VLAN 20 – Classroom – 172.16.20.0/24
- VLAN 30 – Guest – 172.16.30.0/24
- VLAN 67 – Management – 172.16.67.0/28

More details will be added as configuration progresses.
