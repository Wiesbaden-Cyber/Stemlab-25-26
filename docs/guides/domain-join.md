Prerequisites:

A computer running Windows 11 Pro (Home edition cannot join a domain).
Network connectivity to the Domain Controller.
Domain Administrator credentials.

Steps to Join the Computer to the Domain

1. Open Settings

  Click Start
  Select Settings

2. Navigate to System Information
 
  In the left pane, click System
  Scroll down and select About

3. Access Domain Settings

  Under Device specifications, click Domain or workgroup
  In the System Properties window, click Change

4. Join the Domain

  Select the Domain radio button
  Enter the domain name
  Click OK

5. Enter Domain Credentials

  When prompted, enter:
  Username: Administrator
  Password: 1234qwer!@#$QWER

  Click OK.

6. Restart the Computer
  After receiving the welcome message confirming the domain join, click OK
  Restart the computer when prompted
